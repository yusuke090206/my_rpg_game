## 操作方法
一般のゲームと同じくWASDで移動、Eでインベントリを開き所持アイテムの確認ができる。<br>Fキーで自分の立っている座標を中心に調べる、話しかけることができる。<br>会話や持ち物を所持するかの選択士ではY/N(yes/no)で選択できる。<br>ゲームを終わらせたい場合はESCを押すことでタイトルに戻るかゲームを終了できる。

## 写真などの素材
キャラクターの素材はキャラクター合成器を用いて作成しました。<br>マップは生成AI(gemini)を用いて作成しました。

## エンディング条件
**収集できるアイテム一覧**
- 夫婦写真
- 懐かしいパン
- 指輪
- 花
- 古びた鍵
- 新型の拳銃
- 同じ夫婦写真
- 楽譜 <br>
詳しい入手方法はmapsのファイルよりそれぞれのアイテムの座標を確認してください(ネタバレ回避) 

**ネタバレ注意**　　 
- エンディング0 BAD END<br>
一番最初の選択肢でNを選択
- エンディング1 BAD END<br>
全部のアイテムの収集ができていない状態でかつ、拳銃の所持をしていない状態で魔女に話しかける
- エンディング2 HAPPY END<br>
全部のアイテムの収集を完了し、拳銃を所持していない状態で魔女に話しかける
- エンディング3 BAD END<br>
全部のアイテムを所持していない状態でかつ、拳銃を所持している状態で魔女に話しかけ拳銃で撃つ
- エンディング4 SPECIAL END<br>
全部のアイテムを所持している状態でかつ、拳銃を所持している状態で魔女に話しかけ拳銃で撃つ
- エンディング5 HAPPY END<br>
全部のアイテムを所持している状態でかつ、拳銃を所持している状態で魔女に話しかけ拳銃で撃たない
