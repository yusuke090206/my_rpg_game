## 操作方法
一般のゲームと同じくWASDで移動、Eでインベントリを開き所持アイテムの確認ができる。
Fキーで自分の立っている座標を中心に調べる、話しかけることができる。
会話や持ち物を所持するかの選択士ではY/N(yes/no)で選択できる。
ゲームを終わらせたい場合はESCを押すことでタイトルに戻るかゲームを終了できる。

## エンディング条件
ネタバレ注意　　 
- エンディング0 BAD END
　一番最初の選択肢でNを選択
- エンディング1 BAD END
  全部のアイテムの収集ができていない状態でかつ、拳銃の所持をしていない
  状態で魔女に話しかける
- エンディング2 HAPPY END
　全部のアイテムの収集を完了し、拳銃を所持していない状態で魔女に話しかける
- エンディング3 BAD END
　全部のアイテムを所持していない状態でかつ、拳銃を所持している状態で魔女に話しかけ
　拳銃で撃つ
- エンディング4 SPECIAL END
　全部のアイテムを所持している状態でかつ、拳銃を所持している状態で魔女に話しかけ
　拳銃で撃つ
- エンディング5 HAPPY END
　全部のアイテムを所持している状態でかつ、拳銃を所持している状態で魔女に話しかけ
　拳銃で撃たない
